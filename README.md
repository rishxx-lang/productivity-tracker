#  Productivity Tracker

A smart task and productivity management web application built using HTML, CSS, and JavaScript. This app allows users to efficiently manage daily tasks, track completion progress, and store data persistently using browser localStorage.

## Features

-  Add new tasks
-  Mark tasks as completed
-  Delete tasks
-  Real-time task statistics (Total & Completed)
-  Data persistence using localStorage
-  Modern premium dark UI design

## Tech Stack

- HTML
- CSS
- JavaScript
- Browser localStorage API

##  Concepts Implemented

- DOM Manipulation
- Event Handling
- Array Operations
- Data Persistence
- Dynamic UI Rendering

##  Purpose

This project was built to strengthen frontend development skills and demonstrate practical JavaScript implementation for real-world task management scenarios.
